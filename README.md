**Nasıl Çalıştırılır?**

1. Klonla
`` $ git clone..``

2. Eksik gem varsa bundle et
``$ bundle install``

3. Veritabanını güncelle
`` $ rake db:migrate``

 `` $ rake db:seed``

4. Sunucuyu çalıştır
`` $ rails s``

5. Web tarayıcısını aç
`` localhost:3000 ``
